# Product Spec

Market management consists some product and customer related data computation and data consistency problems when it is dealt with only human power and human brain. Data persistency and computation is the main purpose of the computers which it can resolve most of the market management data related problems. Despite it can be extended, this application will provide a robust, reliable and scalable solution to following pointed features.

- Product related data management like pricing, stocking etc.
- Sale data management and cascading effects
- Purchase data management and cascading effects
- Data visulations and maybe some data science after all
